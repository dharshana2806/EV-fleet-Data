# 🚗 EV Fleet Management System using Machine Learning

## 📌 Project Overview
This project focuses on analyzing and predicting the revenue generated by Electric Vehicles (EVs) in a fleet management system using Machine Learning. The dataset includes vehicle specifications, battery information, charging details, maintenance records, driver information, and operational metrics.

The project performs:
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning Model Training
- Revenue Prediction
- Performance Evaluation
- Data Visualization

---

## 📂 Dataset

**Dataset Name:** EV Fleet Management Dataset (Custom Generated)

### Features Included
- Vehicle_ID
- Vehicle_Type
- Manufacturer
- Model
- Manufacture_Year
- Battery_Percentage
- Battery_Capacity
- Charging_Time
- Charging_Cost
- Energy_Consumed
- Kilometer
- Maintenance_Cost
- Revenue_Generated *(Target Variable)*
- Driver_Experience
- Vehicle_Status
- Month
- Garage_Status
- Accident_Count
- Overspeed_Violation_Count
- Expected_Travel_Distance
- and other operational features.

---

## 🎯 Target Variable

**Revenue_Generated**

The Machine Learning model predicts the revenue generated by an EV based on vehicle characteristics, battery status, charging details, maintenance, and operational information.
